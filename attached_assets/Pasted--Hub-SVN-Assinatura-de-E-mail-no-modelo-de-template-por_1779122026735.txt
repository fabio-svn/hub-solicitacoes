# Hub SVN — Assinatura de E-mail no modelo de template por variante + botão Duplicar

## Contexto

Após análise, ficou claro que Assinatura de E-mail **NÃO** se encaixa no modelo de "variant source no layer" porque cada marca tem:
- Logo com dimensões diferentes (precisa de posicionamento individual)
- Algumas marcas (Gestão/Global/IMB/Agro/Proteção/Wealth) usam layout dual com cargo, outras não

Esses são casos de **layout específico por variante** — mesmo modelo do Cartão Comemorativo. Cada marca vira um template independente.

Na página `/admin/templates`, adicionar botão em cada template **que pertence a um tipo com `template_variant_field`**.

### Comportamento

Ao confirmar:

```ts
async function duplicateToVariant(sourceTemplateId, newVariantValue, newName) {
  const source = await db.query.artTemplates.findFirst({ where: eq(id, sourceTemplateId) });
  
  const newConfig = JSON.parse(JSON.stringify(source.config));
  // Atualizar tipo dentro do config (não muda, mas pra garantir)
  newConfig.tipo = source.tipo;
  
  const [created] = await db.insert(artTemplates).values({
    tipo: source.tipo,
    name: newName,
    variant_value: newVariantValue,
    config: newConfig,
    is_active: false,  // criado inativo, usuário ativa após editar
    updated_by: req.user.id,
  }).returning();
  
  // Redireciona pro editor do novo
  res.json({ id: created.id, redirect: `/admin/templates/${created.id}/edit` });
}
```

### Filtro de variantes disponíveis

O dropdown mostra **apenas variantes que ainda não têm template** pra esse tipo. Lógica:

```ts
const formSchema = FORM_SCHEMAS[source.tipo];
const allOptions = formSchema.fields.find(f => f.name === formSchema.template_variant_field)?.options || [];
const existingVariants = await db.select({ v: artTemplates.variant_value })
  .from(artTemplates)
  .where(eq(artTemplates.tipo, source.tipo));
const existingSet = new Set(existingVariants.map(r => r.v));
const availableOptions = allOptions.filter(opt => !existingSet.has(opt.value));
```

Se `availableOptions.length === 0` (todas variantes já têm template), o botão "Duplicar" aparece desabilitado com tooltip "Todas variantes já têm template".

### Nome sugerido automático

Tentar inferir o sufixo da variante atual e trocar:

```ts
// Se template atual chama "Assinatura SVN Investimentos" e ele tem variant_value='svn-investimentos':

const sourceVariantLabel = options.find(o => o.value === source.variant_value)?.label;

const suggestedName = source.name.replace(sourceVariantLabel, newVariantLabel) || `${source.name} (${newVariantLabel})`;
```

Esse nome é só sugestão — o usuário pode editar antes de confirmar.
