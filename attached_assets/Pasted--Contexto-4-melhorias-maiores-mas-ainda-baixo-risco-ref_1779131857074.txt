## Item 1 — Materiais detalhes declarativo (form-eventos)

Em public/form-eventos.html, há ~100 linhas de IIFE coletando 
sub-fields de cada material com schema diferente.

Ação:
1. No topo do arquivo (ou em config.js se quiser global), definir 
   schema declarativo:

   const MATERIAIS_DETALHES_SCHEMA = {
     'pacote-personalizado': {
       fields: [
         { key: 'personConvite', source: 'input', id: 'matPersonConvite' },
         { key: 'personPagina', source: 'input', id: 'matPersonPagina' },
       ],
     },
     'banner-impresso': {
       fields: [
         { key: 'tamanho', source: 'input', id: 'matBannerTam' },
         { key: 'conteudo', source: 'input', id: 'matBannerConteudo' },
       ],
     },
     // ... os outros 8 materiais (flyer, brindes-personalizados, 
     //     captacao-audiovisual, instagram, email-marketing, 
     //     apoio-live-online, pagina-sorteio, projeto-stand)
   };

2. Substituir a IIFE de ~100 linhas por uma função genérica:

   function buildMateriaisDetalhes(materiaisSelecionados) {
     const det = {};
     for (const matId of materiaisSelecionados) {
       const schema = MATERIAIS_DETALHES_SCHEMA[matId];
       if (!schema) continue;
       const d = {};
       for (const field of schema.fields) {
         let val;
         if (field.source === 'input') {
           val = document.getElementById(field.id)?.value?.trim();
         } else if (field.source === 'radio') {
           val = document.querySelector(`input[name="${field.name}"]:checked`)?.value;
           if (val && field.transform) val = field.transform(val);
         }
         if (val !== undefined && val !== '') d[field.key] = val;
       }
       if (Object.keys(d).length) det[matId] = d;
     }
     return Object.keys(det).length ? det : undefined;
   }

3. Validar que o output é IDÊNTICO ao da IIFE original (testar com 
   diferentes combinações de materiais selecionados).

## Item 2 — IBGE cache + error UI

Em public/form-eventos.html, IBGE é chamado direto sem cache. Sem 
error UI também.

Ação:
1. Criar public/ibge-loader.js:

   const IBGE_CACHE_KEY = 'svn_ibge_v1';
   const IBGE_CACHE_TTL_MS = 24 * 60 * 60 * 1000;

   const IBGELoader = {
     async loadEstados() {
       try {
         const cached = JSON.parse(localStorage.getItem(IBGE_CACHE_KEY) || 'null');
         if (cached && cached.estados && Date.now() - cached.timestamp < IBGE_CACHE_TTL_MS) {
           return cached.estados;
         }
       } catch {}
       
       const res = await fetch('https://servicodados.ibge.gov.br/api/v1/localidades/estados?orderBy=nome');
       if (!res.ok) throw new Error('ibge_estados_failed');
       const estados = await res.json();
       
       try {
         localStorage.setItem(IBGE_CACHE_KEY, JSON.stringify({ estados, timestamp: Date.now() }));
       } catch {}
       
       return estados;
     },
     
     async loadCidades(estadoId) {
       const cacheKey = IBGE_CACHE_KEY + '_cidades_' + estadoId;
       try {
         const cached = JSON.parse(localStorage.getItem(cacheKey) || 'null');
         if (cached && cached.cidades && Date.now() - cached.timestamp < IBGE_CACHE_TTL_MS) {
           return cached.cidades;
         }
       } catch {}
       
       const res = await fetch(`https://servicodados.ibge.gov.br/api/v1/localidades/estados/${estadoId}/municipios?orderBy=nome`);
       if (!res.ok) throw new Error('ibge_cidades_failed');
       const cidades = await res.json();
       
       try {
         localStorage.setItem(cacheKey, JSON.stringify({ cidades, timestamp: Date.now() }));
       } catch {}
       
       return cidades;
     }
   };

2. Em form-eventos.html, importar ibge-loader.js via <script src>. 
   Substituir as chamadas diretas de fetch IBGE por IBGELoader.

3. Error UI: se loadEstados ou loadCidades lançar erro, mostrar no 
   dropdown:

   sel.innerHTML = '<option value="">Erro ao carregar — clique pra tentar novamente</option>';
   sel.onchange = () => loadEstados();  // retry
