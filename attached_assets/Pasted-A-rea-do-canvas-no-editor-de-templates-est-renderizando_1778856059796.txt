A área do canvas no editor de templates está renderizando em tamanho natural, excedendo o viewport. Adicionar fit-to-view automático e controles de zoom:
1. Fit-to-view por padrão
Quando um template é carregado (ou ao redimensionar a janela), calcular um canvasScale inicial que faz o canvas inteiro caber no viewport disponível (descontando toolbar superior + bottom panel + padding). Usar a menor dimensão limitante:
jslet canvasScale = 1;

function fitToView() {
  const scrollEl = document.getElementById('canvasScroll');
  const availW = scrollEl.clientWidth - 48;
  const availH = scrollEl.clientHeight - 48;
  const sW = availW / currentTemplate.canvas.width;
  const sH = availH / currentTemplate.canvas.height;
  canvasScale = Math.min(sW, sH, 1);
  applyCanvasScale();
}

function applyCanvasScale() {
  const wrap = document.getElementById('canvasWrap');
  wrap.style.width = (currentTemplate.canvas.width * canvasScale) + 'px';
  renderCanvas();
}
Chamar fitToView() em:

loadTemplate() (logo após renderizar o canvas)
Listener de window.resize

2. Controles de zoom na toolbar do canvas
Adicionar entre o título e o toggle Editar/Preview:
html<button onclick="zoomOut()" title="Reduzir">−</button>
<span id="zoomLevel" style="font-size:0.78rem;color:#aaa;min-width:48px;text-align:center">35%</span>
<button onclick="zoomIn()" title="Ampliar">+</button>
<button onclick="fitToView()" title="Ajustar à tela">Ajustar</button>
Funções:
jsfunction zoomIn()  { canvasScale = Math.min(canvasScale + 0.1, 2); applyCanvasScale(); updateZoomLabel(); }
function zoomOut() { canvasScale = Math.max(canvasScale - 0.1, 0.1); applyCanvasScale(); updateZoomLabel(); }
function updateZoomLabel() {
  document.getElementById('zoomLevel').textContent = Math.round(canvasScale * 100) + '%';
}
E chamar updateZoomLabel() dentro do applyCanvasScale() também (pra atualizar quando vier do fit).
3. Garantir que coordenadas de layers continuam corretas
A função getScale() atual já calcula wrap.offsetWidth / canvas.width, e como applyCanvasScale() seta wrap.style.width, o offsetWidth reflete o tamanho atual. Coordenadas de drag/resize/posicionamento de boxes vão funcionar sem ajuste extra.
Conferir que a img.bg-img continua com width: 100%; height: auto no CSS pra acompanhar o wrap.
4. Persistir zoom entre templates (opcional, recomendado)
Salvar canvasScale em sessionStorage pra preservar a preferência ao trocar entre templates ou recarregar a página.
