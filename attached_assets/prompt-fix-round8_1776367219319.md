## 2. `dashboard.html` — Formatar datas no drawer

### 2a. Adicionar função `formatarData()` no script

Adicionar junto às outras funções utilitárias (`dataRelativa`, `periodoGrupo`):

```js
function formatarData(raw) {
  if (!raw) return raw;
  // Formato YYYY-MM-DD → "01 de maio de 2026"
  const meses = ['janeiro','fevereiro','março','abril','maio','junho',
                 'julho','agosto','setembro','outubro','novembro','dezembro'];
  // Tentar parse como data pura (YYYY-MM-DD)
  const matchData = String(raw).match(/^(\d{4})-(\d{2})-(\d{2})$/);
  if (matchData) {
    const d = parseInt(matchData[3]);
    const m = parseInt(matchData[2]) - 1;
    const a = matchData[1];
    return `${d} de ${meses[m]} de ${a}`;
  }
  // Tentar parse como datetime ISO
  const d = new Date(raw);
  if (!isNaN(d.getTime())) {
    return `${d.getDate()} de ${meses[d.getMonth()]} de ${d.getFullYear()}`;
  }
  return raw;
}
```
