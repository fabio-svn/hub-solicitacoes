Listas do ClickUp

← Admin
Como usar: cole o ID numérico da lista do ClickUp (na URL: app.clickup.com/…/v/l/LISTA_ID — o número longo do meio). Clique em Validar para confirmar que o ID existe e o token tem acesso, e em Salvar para gravar (o servidor revalida antes). Vários formulários podem usar a mesma lista. Campo vazio + Salvar remove a configuração e o tipo volta a usar a lista padrão.
