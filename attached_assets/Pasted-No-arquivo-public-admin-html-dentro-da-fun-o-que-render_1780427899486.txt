No arquivo public/admin.html, dentro da função que renderiza o gráfico de pizza (linha ~748), o canvas está sendo desenhado sem ajuste de devicePixelRatio. Em telas Retina/HiDPI, isso causa renderização borrada.

LOCALIZAR:

        const CORES = ['#AC3631','#221B19','#4a7c2f','#1e40af','#9333ea','#ea580c','#0891b2','#be123c','#854d0e','#166534'];
        canvas.width = 200; canvas.height = 200;
        const ctx = canvas.getContext('2d');
        const cx = 100, cy = 100, r = 85, rInner = 50;

SUBSTITUIR POR:

        const CORES = ['#AC3631','#221B19','#4a7c2f','#1e40af','#9333ea','#ea580c','#0891b2','#be123c','#854d0e','#166534'];
        const DPR = window.devicePixelRatio || 1;
        const SIZE = 200;
        canvas.width = SIZE * DPR;
        canvas.height = SIZE * DPR;
        canvas.style.width = SIZE + 'px';
        canvas.style.height = SIZE + 'px';
        const ctx = canvas.getContext('2d');
        ctx.scale(DPR, DPR);
        const cx = 100, cy = 100, r = 85, rInner = 50;

EXPLICAÇÃO: o canvas agora tem resolução interna 400x400 em Retina (DPR=2), mas é exibido em 200x200 CSS pixels. O ctx.scale(DPR, DPR) faz com que todas as coordenadas de desenho continuem em CSS pixels (cx=100, r=85, etc.) — não precisa mudar o resto do código.

VERIFICAR DEPOIS:
1. Recarregar /admin no browser
2. O gráfico de pizza deve aparecer NÍTIDO (sem borrão nas bordas)
3. Testar em diferentes telas: Retina (Mac), monitor externo HiDPI, monitor 1080p comum

⚠️ ATENÇÃO: o branch do total=0 tem clearRect(0, 0, canvas.width, canvas.height) com canvas.width agora sendo 400 em Retina. Isso ainda funciona (limpa mais área do que precisa) mas pode causar warning em algum browser. Se quiser ser preciso, substituir também:

LOCALIZAR (no branch total === 0):
          const ctx = canvas.getContext('2d');
          ctx.clearRect(0, 0, canvas.width, canvas.height);

SUBSTITUIR POR:
          const ctx = canvas.getContext('2d');
          ctx.clearRect(0, 0, canvas.width, canvas.height);
          // OBS: nesse branch o canvas ainda não foi resized; clearRect funciona como antes

(Na verdade esse branch fica antes do canvas.width/height ser setado, então usa as dimensões originais do HTML (200x200). Pode deixar como está.)
