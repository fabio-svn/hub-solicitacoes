Como o Replit agora possui um Agente de IA integrado (Replit Agent), você pode dar um comando direto para ele ler seus arquivos e aplicar a correção de forma global.

Copie e cole o prompt abaixo no chat do Replit (na lateral direita ou no Replit Agent):

Prompt para o Replit Agent
"Estou com um problema nos meus formulários HTML. Eles deveriam enviar dados e arquivos (uploads) para webhooks do N8N, mas o sistema atual tenta enviar para uma rota interna /api/solicitacoes que não está funcionando, e os arquivos binários (fotos/documentos) não estão sendo incluídos no envio.

Por favor, realize as seguintes alterações em todos os arquivos de formulário (Ex: form-divulgacao-nps.html, form-cartao-visita.html, etc):

Envio Direto: Altere a função submitForm() para que o fetch envie os dados diretamente para a URL do Webhook do N8N correspondente a cada formulário (use as URLs que estão no código ou que eu forneci).

Correção do FormData: Certifique-se de que todos os campos (texto e arquivos) sejam adicionados ao objeto FormData. Para os campos de arquivo (como fotoPerfil), use formData.append('nome_do_campo', input.files[0]).

Mapeamento de IDs: Garanta que as chaves do FormData correspondam exatamente aos nomes esperados pelo N8N (Ex: nome_assessor, email, foto, unidade).

Remoção de JSON.stringify: Não envie os dados dentro de uma string JSON chamada 'dados'. Envie cada campo individualmente no FormData para que o N8N receba como parâmetros de formulário.

Feedback de Erro: Se o fetch falhar ou retornar erro, exiba a mensagem no elemento submitError.

URLs de referência para os arquivos principais:

NPS: https://auto.portalsvn.com.br/webhook/arte-nps (Campo foto: foto)

Cartão Digital: https://auto.portalsvn.com.br/webhook/cartao-digital (Foto: field_4987b4c)

Cartão Visita Físico: https://auto.portalsvn.com.br/webhook/solicitacao-cartao-fisico

Certificados: https://auto.portalsvn.com.br/webhook/gerar-certificados

Verifique também o arquivo utils.js e upload-feedback.js para garantir que a lógica de upload esteja integrada ao submit."
