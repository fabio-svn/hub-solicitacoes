### 3.3 — Simplificar o script posicionarWhatsApp
Em cada um dos 24 HTMLs, LOCALIZAR o script (geralmente perto da linha 
1337+):
```js
function posicionarWhatsApp() {
  const wa = document.querySelector('.float-whatsapp');
  if (!wa) return;
  const outros = [...document.querySelectorAll('.float-home')]
    .filter(el => el !== wa && getComputedStyle(el).display !== 'none');
  if (outros.length === 0) {
    wa.style.bottom = '78px';
  } else {
    const alturas = outros.map(el => parseInt(el.style.bottom || getComputedStyle(el).bottom) || 20);
    const maiorAltura = Math.max(...alturas);
    wa.style.bottom = (maiorAltura + 58) + 'px';
  }
}
```

SUBSTITUIR por versão simplificada (sem dependência de float-home):
```js
function posicionarWhatsApp() {
  const wa = document.querySelector('.float-whatsapp');
  if (wa) wa.style.bottom = '20px';
}
```

Ou se preferir, deletar a função inteira E o script IIFE inteiro que 
chama ela — o WhatsApp já tem `bottom: 20px` definido no CSS via 
`.float-whatsapp` (verificar no style.css).

VERIFICAR em style.css se `.float-whatsapp` tem `bottom` definido. Se 
tiver `bottom: 20px` ou similar, pode DELETAR o script todo (e o IIFE 
que chama posicionarWhatsApp).
