## Contexto

Novo tipo de layer "Shape" no editor de templates, suportando retângulos 
e elipses. Renderiza no canvas, props panel pra editar cor de preenchimento, 
borda, e (para retângulos) border-radius.

Arquivos:
- artifacts/api-server/src/types/art-template.ts
- artifacts/api-server/src/services/template-renderer.ts
- artifacts/api-server/public/admin-templates.html

## Fix 2 — Backend: render de shape no template-renderer.ts

Em src/services/template-renderer.ts:

LOCALIZAR a função `renderImage` (~linha 257). Após ela, ADICIONAR uma 
nova função:

```ts
async function renderShape(
  layer: ShapeLayer,
  composites: sharp.OverlayOptions[],
  bgWidth: number,
  bgHeight: number,
) {
  if (layer.w <= 0 || layer.h <= 0) {
    logger.warn(`[render] shape ${layer.id} skipped: zero/negative dimensions`);
    return;
  }

  const safeW = Math.min(layer.w, bgWidth - layer.x);
  const safeH = Math.min(layer.h, bgHeight - layer.y);
  if (safeW <= 0 || safeH <= 0) return;

  const fill = layer.fill || 'none';
  const stroke = layer.stroke || 'none';
  const strokeWidth = layer.stroke_width || 0;

  // Em SVG, stroke é centrado na borda. Ajustar inset pra ficar todo dentro.
  const inset = strokeWidth / 2;

  let shapeEl: string;
  if (layer.shape === 'ellipse') {
    const cx = safeW / 2;
    const cy = safeH / 2;
    const rx = Math.max(0, (safeW / 2) - inset);
    const ry = Math.max(0, (safeH / 2) - inset);
    shapeEl = `<ellipse cx="${cx}" cy="${cy}" rx="${rx}" ry="${ry}" fill="${fill}" stroke="${stroke}" stroke-width="${strokeWidth}"/>`;
  } else {
    // rectangle
    const x = inset;
    const y = inset;
    const w = Math.max(0, safeW - strokeWidth);
    const h = Math.max(0, safeH - strokeWidth);
    const rx = Math.min(layer.border_radius || 0, w / 2, h / 2);
    shapeEl = `<rect x="${x}" y="${y}" width="${w}" height="${h}" rx="${rx}" ry="${rx}" fill="${fill}" stroke="${stroke}" stroke-width="${strokeWidth}"/>`;
  }

  const svg = `<svg width="${safeW}" height="${safeH}" xmlns="http://www.w3.org/2000/svg">${shapeEl}</svg>`;
  const buffer = await sharp(Buffer.from(svg)).png().toBuffer();

  composites.push({
    input: buffer,
    left: Math.round(layer.x),
    top: Math.round(layer.y),
  });
}
```

GARANTIR que `ShapeLayer` está importado no topo do arquivo:
```ts

```

LOCALIZAR o loop que processa layers (~linha 410):
```ts

if (layer.type === 'text-block') await renderTextBlock(layer, dataStr, composites, renderCtx);

```

ADICIONAR nova linha:

## Fix 5 — Frontend: render do shape no canvas (preview ao vivo)

LOCALIZAR onde as layers são renderizadas no canvas DOM (provavelmente 
função `renderLayerBox(layer)` ou similar — procurar por `.layer-box`).

ADICIONAR caso pra shape:
```js
// Dentro da função que renderiza layer no canvas:
if (layer.type === 'shape') {
  // Render via SVG inline pro preview ficar fiel ao backend
  const fill = layer.fill || 'transparent';
  const stroke = layer.stroke || 'none';
  const sw = layer.stroke_width || 0;
  
  let shapeHtml;
  if (layer.shape === 'ellipse') {
    shapeHtml = `<svg width="100%" height="100%" viewBox="0 0 ${layer.w} ${layer.h}" preserveAspectRatio="none" xmlns="http://www.w3.org/2000/svg">
      <ellipse cx="${layer.w/2}" cy="${layer.h/2}" 
               rx="${layer.w/2 - sw/2}" ry="${layer.h/2 - sw/2}"
               fill="${fill}" stroke="${stroke}" stroke-width="${sw}"/>
    </svg>`;
  } else {
    const rx = Math.min(layer.border_radius || 0, layer.w/2, layer.h/2);
    shapeHtml = `<svg width="100%" height="100%" viewBox="0 0 ${layer.w} ${layer.h}" preserveAspectRatio="none" xmlns="http://www.w3.org/2000/svg">
      <rect x="${sw/2}" y="${sw/2}" 
            width="${layer.w - sw}" height="${layer.h - sw}" 
            rx="${rx}" ry="${rx}"
            fill="${fill}" stroke="${stroke}" stroke-width="${sw}"/>
    </svg>`;
  }
  
  return shapeHtml;
}
```

NOTA: usar o pattern exato do código existente pra renderizar layers no 
canvas. Se for via innerHTML, retornar string. Se for via createElement,

## Critério de validação

1. `pnpm run build` compila sem erros (TypeScript)
2. Abrir editor de qualquer template → painel "+ Adicionar" mostra 
   3 botões: Texto, Imagem, Forma
3. Clicar "+ Forma": nova layer aparece no canvas (retângulo ruby red 
   200x150 em x=50, y=50)
4. Selecionar a layer → painel direito mostra props:
   - Tipo (retângulo/elipse)
   - X, Y, W, H
   - Preenchimento (com swatches)
   - Cor borda (com swatches)
   - Espessura borda
   - Arredondamento (só se retângulo)
5. Mudar tipo pra "Elipse": canvas atualiza, props "Arredondamento" some
6. Arrastar a shape no canvas: move como qualquer layer
7. Salvar template → recarregar editor → shape persiste
8. Gerar uma solicitação real do tipo que usa esse template: a shape 
   aparece na arte renderizada (PNG/PDF final)
9. Borda com espessura > 0 + cor: visível no canvas e na arte
10. Retângulo com border_radius > 0: cantos arredondados visíveis

## Não faça

- Não tocar nos types existentes (TextLine, TextBlock, Image) — só adicionar 
  ShapeLayer ao union
- Não criar arquivo novo de service — usar template-renderer.ts existente
- Não tocar no pdf-renderer.ts — ele só faz overlay PNG no PDF, vai pegar 
  o shape automaticamente do PNG gerado
- Não adicionar suporte a outros shapes (triangle, polygon) — escopo é 
  apenas rectangle e ellipse
- Se renderColorPickerWithSwatches não existir (Prompt 10 não aplicado), 
  usar `<input type="color">` simples — não falhar
