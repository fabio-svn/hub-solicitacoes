## 3. Indicador de subtipo nos forms com escolha na step 1

### Padrão visual a implementar:

Após selecionar o subtipo na step 1 e avançar para a step 2,
o título do form deve mostrar o subtipo selecionado:

```html
<!-- Exemplo de como deve ficar o h2 no topo do step 2: -->
<h2 style="font-weight:500;font-size:1.1rem;margin:0" id="formTitulo">
  Eventos <!-- será atualizado via JS para "Eventos — Presencial" -->
</h2>
```

```js
// Função reutilizável para atualizar o título:
function atualizarTituloForm(base, subtipo) {
  const el = document.getElementById('formTitulo');
  if (el) el.textContent = subtipo ? base + ' — ' + subtipo : base;
}
```
