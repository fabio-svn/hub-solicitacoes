Fix 1 — Bug de precedência no font-weight do preview + fontes "Light"
Hoje o overlay usa:
font-weight:${layer.font_weight || layer.bold ? '700' : '400'}
Por precedência isso vira (font_weight || bold) ? '700' : '400', então
QUALQUER font_weight definido (ex.: 300 Light) renderiza como 700 no preview.
Além disso, 'Taviraj Light'/'Nunito Sans Light' não existem como fonte
carregada no navegador (caem em sans-serif). Vamos corrigir as duas coisas só
no preview do overlay (sem alterar o font_family salvo).
LOCALIZAR:
js          txtPreview.style.cssText = [
            'position:absolute;inset:0;pointer-events:none;overflow:hidden;',
            `font-family:${layer.font_family ? `'${layer.font_family}',` : ''}sans-serif;`,
            `font-size:${scaledSize}px;`,
            `color:${layer.color || '#000'};`,
            `font-weight:${layer.font_weight || layer.bold ? '700' : '400'};`,
SUBSTITUIR por:
js          // Mapeia nomes "Light" (que existem como arquivo no servidor, mas não
          // como fonte carregada no navegador) → família carregada + peso, APENAS
          // para o preview do overlay. NÃO altera o layer.font_family salvo.
          const _fontMap = {
            'Taviraj Light':     { fam: "'Taviraj'",     w: 300 },
            'Nunito Sans Light': { fam: "'Nunito Sans'", w: 300 },
          };
          const _pf = _fontMap[layer.font_family] || { fam: layer.font_family ? `'${layer.font_family}'` : '', w: 400 };
          const _weight = layer.font_weight || (layer.bold ? 700 : _pf.w);
          txtPreview.style.cssText = [
            'position:absolute;inset:0;pointer-events:none;overflow:hidden;',
            `font-family:${_pf.fam ? _pf.fam + ',' : ''}sans-serif;`,
            `font-size:${scaledSize}px;`,
            `color:${layer.color || '#000'};`,
            `font-weight:${_weight};`,

Fix 2 — CSS inválido em .no-template
LOCALIZAR:
css    .no-template { display: flex; align-items: center; justify-content: flex: 1; color: rgba(255,255,255,0.3); font-size: 0.9rem; }
SUBSTITUIR por:
css    .no-template { display: flex; align-items: center; justify-content: center; flex: 1; color: rgba(255,255,255,0.3); font-size: 0.9rem; }
(justify-content: flex: 1; é uma declaração quebrada que o browser ignora;
o que se queria era justify-content: center + flex: 1.)
