Itens selecionados
1
Tenho a maioria das informações

Quando as informações mais básicas do evento estiverem pré-definidas.

Ex.: Tenho Nome, Data, Local, Formato, etc

2
Tenho algumas informações

Quando algumas informações estiverem definidas, mas você precisa de apoio para fechar outros pontos.

Ex.: Quero realizar uma palestra sobre FIIs no dia 7, mas ainda preciso definir o local.

3
Ainda estou estruturando

Quando você tiver uma ideia para um evento, mas precisa de apoio para estruturar.

Ex.: Gostaria de realizar uma experiencia com clientes, mas preciso de assistência.

Para que o processo ocorra de forma eficiente, escolha a opção que mais se adequa a realidade do seu evento. Quanto mais informações forem preenchidas, mais rápido será o processo.

Esses prazos garantem qualidade na entrega e organização adequada. Solicitações fora do prazo mínimo podem não ser atendidas ou terão escopo reduzido.
