1

Quando as informacoes mais basicas do evento estiverem pre-definidas.

Ex.: Tenho Nome, Data, Local, Formato, etc

2
Tenho algumas informacoes

Quando algumas informacoes estiverem definidas, mas voce precisa de apoio para fechar outros pontos.

Ex.: Quero realizar uma palestra sobre FIIs no dia 7, mas ainda preciso definir o local.

3
Ainda estou estruturando

Quando voce tiver uma ideia para um evento, mas precisa de apoio para estruturar.

Ex.: Gostaria de realizar uma experiencia com clientes, mas preciso de assistencia.

Para que o processo ocorra de forma eficiente, escolha a opcao que mais se adequa a realidade do seu evento. Quanto mais informacoes forem preenchidas, mais rapido sera o processo.
