Como usar: cole o ID numérico da lista do ClickUp (encontrado na URL: app.clickup.com/…/v/l/LISTA_ID). Clique em Validar para confirmar que o ID existe e o token tem acesso. Clique em Salvar para gravar. O servidor revalida antes de salvar. Vários formulários podem usar a mesma lista. Campo vazio + Salvar remove a configuração e faz o tipo voltar a usar a lista padrão.
