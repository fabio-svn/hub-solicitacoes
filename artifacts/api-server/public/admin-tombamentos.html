Tombamentos

Geração em massa de assinaturas e cartões digitais a partir de uma planilha.

A marca vale para todo o tombamento — todas as assinaturas e cartões usarão o logo dela.

Carregando…

Formatos: .xlsx, .xls ou .csv (máx. 15 MB). Colunas: nome, e-mail, telefone, cargo, tem CFP, arquivo da foto. (A marca não vai na planilha — é a do tombamento.) A coluna arquivo da foto é opcional: sem ela, o sistema casa as fotos pelo nome.

Cartões digitais: suba um .zip com as fotos (máx. 100 MB) e o sistema casa cada pessoa pelo nome — você confere antes de gerar. A coluna arquivo_foto é opcional (se preenchida, tem prioridade). Disponível apenas para marcas com cartão digital (Investimentos, Capital, Connect).

Revisão de fotos

Confira os pares antes de gerar. revisar = casou só pelo primeiro nome ou está ambíguo, vale conferir.
